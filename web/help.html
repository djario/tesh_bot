Setup

  1. lade den Bot per Link ein
  2. führe /setup aus und gebe eine Rolle an, die Einstellungen am Bot vornehmnen kann
    für die meisten Module können neue Rollen definiert werden
  3. jetzt füge mit /channel add alle Kategorien hinzu die Roleplay-Channel enthalten
  4. Nun ist der Bot startklar
    beim aktivieren von Modulen erst den entsprechenden Setup-Command ausführen

Module

Der Bot ist Modular aufgebaut, dh. man kann Features die nicht nur einen einzelnen Command beinhalten einzeln aktivieren oder deaktivieren. Dafür stehen bei allen Modulen ein /power Command zur verfügung sowie ein /setup Command.
Es wird dran gearbeitet Einstellungen sowie das aktivieren/deaktivieren von Modulen auf dieser Website zur verfügung zu stellen.

Level

/level
Das Level-Module beinhaltet, dass Nachrichten aus Channeln ausgewählter Kategorien je nach Wortmenge eine Anzahl an Xp erzeugen, welche den User aufleveln lasssen. Es können dann für ereichte Level spezielle Nachrichten oder Rollen festgelegt werden. Für besondere Events können rollen oder zeitweise Boosts erstellt werden, welche die erhaltenen Xp um einen Faktor multiplizieren. Level und Xp können per Command eingesehen werden in vorm einer Level Card. Der Ping, welchen es bei jedem neuen Level gibt kann man ausstellen, falls dies dem User zu nervig sein sollte.

Words

/words
Das Words-Modul beinhaltet einen Wörter Zähler für jede Nachricht in ausgewählten Kategorien. Nachrichten die mit ( starten oder mit ) aufhören werden ignoriert. Wenn der bereich der Wörter innerhalb des festgelegtem Bereiches liegt wird eine Nachricht mit informationen in einen entsprechenden Log-Channel geschickt.

Rooms

/rooms
Der Bot unterstützt eine einfache Zimmer oder Wohnort verfaltung. Es werden Listen erstellt in die sich die User selbständig eintragen können, aber zum austragen ein Teammitglied informieren müssen (um heimliches wechseln zu vermeiden). Je nach freiem Platzt wird der User dann zugeteilt.

Tickets

/ticket
Auserdem unterstützt der Tesh Bot ein simples Ticket System. Es können Pannel mit eigenem Text erstellt werden, bei denen dann neue Private Channel erstellt werden in der zuvor ausgewählten Kategorie. Die Nachrichten in diesem Channel werden beim Schliesßen zusammen in eine txt Datei geschrieben und im Log-Channel für dieses Modul gespeichert.
